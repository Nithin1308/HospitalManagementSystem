# Hospital Management System

A full-stack Hospital Management System developed using:

- Java
- JSP
- Servlets
- MySQL
- Bootstrap
- Maven
- Tomcat
- Docker

## Features

- Patient Management
- Doctor Management
- Appointment Booking
- Billing System
- PDF Bill Generation
- Admin Dashboard
- Login Authentication

## Technologies Used

- Java
- JSP & Servlet
- MySQL
- Bootstrap 5
- Apache Tomcat
- Maven
- Docker
- Render Deployment

## Deployment

Project deployed on Render.

## Setup Instructions

### Clone Repository

```bash
git clone https://github.com/Nithin1308/HospitalManagementSystem.git
```

### Run MySQL

Import the SQL file from:

```text
database/hms.sql
```

### Configure Database

Update DB credentials inside:

```text
DBConnection.java
```

### Run Project

```bash
mvn clean package
```
